# 🎯 PocketOption Pro Signal Bot

بوت تليجرام احترافي لتحليل الأسواق وتوليد إشارات التداول على منصة Pocket Option.

---

## ✨ المميزات

### 📊 التحليل الفني المتقدم
- **12+ مؤشر فني**: RSI, MACD, EMA, Bollinger Bands, Stochastic, CCI, Williams %R, ADX, Parabolic SAR, Ichimoku, Price Action, Candle Patterns
- **تحليل مستويات الدعم والمقاومة** الديناميكية
- **تحليل قوة الاتجاه** (Trend Strength)
- **نظام تقييم جودة الإشارات** (Grade A/B/C)
- **فلترة متقدمة** لتقليل الإشارات الخاطئة

### 💰 إدارة المخاطر
- حساب حجم الصفقة المقترح تلقائياً
- نظام مارتينجيل محسّن (وحدة واحدة فقط)
- إيقاف تلقائي عند تجاوز حدود الخسارة
- تتبع الخسائر المتتالية
- 3 مستويات مخاطرة (منخفض/متوسط/عالي)

### 📈 الإحصائيات والتتبع
- سجل كامل للصفقات
- إحصائيات يومية وإجمالية
- أداء كل زوج عملات على حدة
- تقرير يومي تلقائي
- نسبة نجاح حسب درجة الإشارة

### ⚙️ إعدادات مخصصة لكل مستخدم
- عدد التأكيدات المطلوبة
- فترة الإشارات التلقائية
- مستوى المخاطرة
- رأس المال

## 🚀 التشغيل

### المتطلبات
- Python 3.9+
- توكن بوت تليجرام (من [@BotFather](https://t.me/BotFather))

### الخطوة 1: إنشاء البوت
1. افتح [@BotFather](https://t.me/BotFather) على تليجرام
2. أرسل `/newbot`
3. اختر اسم للبوت
4. انسخ التوكن

### الخطوة 2: التثبيت

```bash
# استنساخ المشروع
git clone <repo_url>
cd pocket_option_bot

# تثبيت المتطلبات
pip install -r requirements.txt
```

### الخطوة 3: الإعداد

```bash
# تعيين توكن البوت (مطلوب)
export TELEGRAM_BOT_TOKEN="YOUR_BOT_TOKEN_HERE"

# اختياري: مفاتيح API إضافية لبيانات أفضل
export ALPHA_VANTAGE_KEY="your_key"    # مجاني من alphavantage.co
export TWELVE_DATA_KEY="your_key"      # مجاني من twelvedata.com
```

### الخطوة 4: التشغيل

```bash
python3 main.py
```

## ⚠️ تنبيه مهم

- هذا البوت أداة مساعدة للتحليل الفني فقط
- **لا يضمن الأرباح** بأي شكل
- التداول في الخيارات الثنائية يحمل مخاطر عالية
- تداول بمسؤولية ولا تخاطر بأكثر مما تتحمل خسارته
- البيانات المستخدمة من مصادر مجانية وقد لا تكون مطابقة 100% لأسعار Pocket Option OTC

## 📝 ملاحظات تقنية

- البوت يستخدم SQLite محلياً أو PostgreSQL (عبر `DATABASE_URL`)
- التخزين المؤقت (Cache) لمدة 60 ثانية لتقليل الطلبات
- يدعم عدة مستخدمين بإعدادات مستقلة
- نظام Fallback ذكي بين مصادر البيانات
- لا يستخدم بيانات وهمية - إذا فشلت جميع المصادر يُعلم المستخدم
